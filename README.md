# Free n8n Invoice Reminder Workflow

A free, ready-to-adapt n8n workflow for small-business operators who track invoices in Google Sheets and want automatic reminder emails before invoices become overdue.

## What it does

- Reads an invoice tracker from Google Sheets
- Filters invoices that are unpaid and due soon
- Sends a friendly reminder email
- Marks the row with the reminder timestamp

## Import

Download `free-n8n-invoice-reminder-workflow.json` and import it into n8n. Replace placeholder sheet IDs, column names, credentials, and sender details before activating.
